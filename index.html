GRIP

A 3D-printed prosthetic hand controlled by brain waves

Initially, I planned to build a prosthetic hand that can be controlled by your thoughts. Using a device called an EEG, I'm able to read different focus levels (calm to hyperfocus), and translate this into code that an arduino can understand.

Unfortunately, I'm not in a positionw where I can invest much into this project - so I've only been able to use low resolution EEGs, meaning that the brain signal readings are a little off.

Because of this, I pivoted and worked on building a robotic hand that mimics my movemnts using computer vision and Google's MediaPipe.
