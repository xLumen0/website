Neuroprosthetic

A 3D-printed prosthetic hand controlled by brain waves

I built a prosthetic hand using 3D printing and EEG technology to interpret brain signals. This project combines my passion for robotics and brain-computer interfaces, aiming to create affordable assistive devices.

Key features include real-time signal processing and a lightweight design. I’m currently refining the EEG accuracy and exploring applications in rehabilitation.
